# Модуль "Visualizer" (Симулятор генерации изображений)

Этот модуль представляет собой **симуляцию** работы ИИ-агента, который "генерирует" изображение комнаты на основе предоставленных ассетов (картинок).

## 1. Назначение

Модуль сканирует папку `assets`, находит в ней изображения (`.png`), а затем запускает "агента", который имитирует долгую обработку и вместо реальной картинки создает текстовое описание результата в файле `result.md`.

Это позволяет протестировать и отладить пайплайн обработки данных без необходимости подключения настоящей тяжелой модели для генерации изображений.

## 2. Как использовать

1.  **Добавьте ассеты.** Поместите несколько `.png` файлов в директорию `visualizer/assets/`. Например:
    *   `wallpaper_classic.png`
    *   `floor_dark_wood.png`
    *   `bed_modern.png`

    *(Вы можете использовать любые изображения, но для лучшего результата в именах файлов должны содержаться слова `wallpaper`, `floor` или `bed`)*.

2.  **Запустите главный скрипт.** Выполните команду из **корневой директории проекта**:
    ```bash
    python visualizer/main.py
    ```

3.  **Проверьте результат.** После выполнения скрипта в папке `visualizer` появится файл `result.md` с текстовым описанием "сгенерированного" изображения.
